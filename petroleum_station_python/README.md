# Petroleum Station Management System

A premium, glassmorphism-inspired Django web application for managing petroleum station operations, including fuel inventory, sales, car wash services, and loyalty rewards.

## 🚀 Features

- **Personalized Dashboards**: Role-based views for Admin, Accountant, Staff, and Customers.
- **Fuel Inventory Management**: Track fuel levels in tanks, record deliveries, and update prices in real-time.
- **Sales Tracking**: Log fuel sales with automatic stock deduction.
- **Car Wash Services**: Manage service bookings and availability.
- **Loyalty Program**: Reward customers with points and redemptions for their purchases.
- **Multi-Stage Approval Workflow**: Secure reporting system where staff reports are reviewed by Accountants and finalized by Admins.
- **Premium UI**: Modern Glassmorphism design system using Bootstrap 5 and customized CSS.

## 🛠️ Tech Stack

- **Backend**: Django (Python)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3 (Vanilla + Glassmorphism), Bootstrap 5.3, Bootstrap Icons
- **Security**: Django Authentication, Role-Based Access Control (RBAC), CSRF Protection

## 📋 Role-Based Access

- **Admin**: Full system access, final approvals, user management.
- **Accountant**: Financial overviews, operational approvals for reports, price management.
- **Staff Member**: Recording sales, bookings, and deliveries; submitting reports.
- **Customer**: View own purchase history, available services, and loyalty rewards.
