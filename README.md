# Petroleum Station Management System - Polyglot Edition

Welcome to the **Petroleum Station Management System**. This repository is organized into two main implementation stacks to support different development needs and features.

## 📂 Repository Structure

### 🐘 [PHP Implementaton](./petroleum_station_php/)
The original, fully functional web management platform.
- **Stack**: PHP 7.4+, MySQL, Bootstrap 5.
- **Features**: Authentication, Station Management, Employee/Customer Records, Real-time Sales, Fuel Inventory Tracking.
- **Status**: Stable & Production-ready.

### 🐍 [Python Implementation](./petroleum_station_python/)
The upcoming Python-based extension and alternative implementation.
- **Stack**: Python 3.x, Flask/Django (planned).
- **Goal**: To implement advanced analytics, AI integration, and automated reporting services.
- **Status**: Under Development.
